Role Name
=========

Ansible role used to install nodejs on an ubuntu based system (tested on debian 10, ubuntu 18.04 and linux mint 19).

Requirements
------------

None.

Role Variables
--------------

Use the following variable to override nodejs version (defaults to 12.x)
- nodejs_version
